## Developing

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
git clone https://github.com/tendieterminal/terminal.git # or clone your own fork
cd src
yarn install
yarn start
```

## Deploying

In the gh-pages branch:

```sh
yarn build
```
